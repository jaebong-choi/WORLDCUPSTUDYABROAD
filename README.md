# 이미지 해체추출 · 이식 · 합성 스킬

레퍼런스 이미지 1장을 올리면 **그 사진을 그대로 재현하는 프롬프트**를 뽑아주는 스킬입니다.
인물·제품·풍경·음식·사물 무엇이 와도 작동하고, 결과를 내 모델·내 제품에 바로 이식할 수 있게 설계돼 있습니다.

## 무엇을 해주나

- 올린 레퍼를 **레이어로 해체**(구도·카메라·조명·색·질감·배경) → 그대로 재현하는 완성형 프롬프트
- **인물 컨셉샷**은 내 모델 초상화와 함께 쓰면 **얼굴만 내 모델로** 바뀌고 컨셉은 그대로
- **인물 + 배경 2장**을 주면 조명·색온도·그림자까지 맞춰 "원래 거기 있던 사람"처럼 합성 + 의상 세팅
- 출력은 항상 4블록: ① 영어 실사용본 ② 한국어 검증 해석 ③ 미감만 가져온 카피 회피 버전 ④ 활용 예시 4가지

## 쓰는 법 (둘 중 택1)

**A. 스킬로 설치해서 쓰기** (스킬 기능을 쓸 수 있는 플랜)
1. Claude 설정 → Capabilities(기능) → 스킬 추가
2. 이 폴더의 `SKILL.md`(또는 동봉된 스킬 파일)를 업로드
3. 채팅에 이미지를 올리고 "추출해줘"라고 입력

**B. 프롬프트 가이드로 쓰기** (스킬 기능이 없어도 가능)
1. `SKILL.md`를 열어 규칙을 복사
2. 쓰는 이미지 생성 도구/챗에 붙여넣고, 레퍼 이미지를 함께 제시
3. "이 규칙대로 이 사진을 해체해서 프롬프트 뽑아줘"라고 요청

## 출력 예시 구조

```
① English (ready to use)   — 복붙해서 바로 생성에 쓰는 영어 프롬프트
② 한국어 검증 해석          — 원본에서 뭘 읽고 어떻게 반영했는지 문장으로
③ Inspired version         — 분위기만 가져오고 고유 요소는 변주(표절 회피)
④ 활용 예시 4가지           — 피사체·배경·조명·구도 바꾸는 패치
```

## 참고

- 프롬프트에 모델 종속 문법(미드저니 기호 등)을 넣지 않고, 화면비·해상도는 사용자가 직접 정하는 방식입니다.
- 라벨·로고 등 타인 저작 요소는 그대로 복제하지 않습니다.
- 퍼스널컬러 등 사진만으로 단정하기 어려운 항목은 "추정"으로 표기됩니다.
